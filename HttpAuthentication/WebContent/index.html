Simple http basic authenticator service

Used from DataPower or API Management

URL: "http://localhost:8080/HttpAuthentication/HttpBasicAuthenticator"

The service checks for the Authorization header.

If the header is present it check for credentials.

Any combination of user and password is authenticated (http 200)

the only exception is:

user: john

pwd: wrong

In this case the service returns Forbidden (http 403)

** Working with Tomcat 7 **
